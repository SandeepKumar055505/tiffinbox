EVENT BUS WIRING (pg-boss) — What triggers what
PAYMENT_SUCCESS
→ generate meal_cells job
→ send confirmation email
→ create ledger entry
→ update subscription state
DELIVERY_FAILED
→ auto-credit ledger
→ notify user
→ mark SLA breach
→ admin alert job
PLAN_EXPIRING (2d before)
→ send renewal reminder
→ show monthly unlock (if eligible)
→ streak preservation nudge
MEAL_SKIPPED
→ update ledger (if auto)
→ break streak check
→ notify user of credit
→ kitchen ops update
NIGHTLY_CRON (11:30 PM)
→ generate tomorrow's meals
→ streak check all users
→ kitchen prep report
→ menu preview notifications
TICKET_CREATED
→ admin dashboard badge
→ auto-tag by keyword
→ priority escalation if CRITICAL
→ admin email alert
